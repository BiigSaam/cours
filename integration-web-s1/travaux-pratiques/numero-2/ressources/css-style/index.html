Consignes de la pratique (cliquer pour afficher / cacher)

  1. Ecrire le contenu des attributs "style" des balises dans la balise <main> dans une balise <style> que vous mettrez dans la balise <head> avec les sélecteurs associés
  2. Supprimer ensuite les attributs "style" et leur valeurs
  3. Sauvegarder le fichier (Ctrl/command (⌘) s) puis consultez votre page sur votre navigateur
