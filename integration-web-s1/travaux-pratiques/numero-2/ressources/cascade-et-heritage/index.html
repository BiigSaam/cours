Consignes de la pratique

(cliquer pour afficher)
  • Ajouter les déclarations et sélecteurs CSS de façon à ce que le code corresponde aux critères définit pour chaque bloc et ce sans modifier le code déjà présent dans le fichier style.css. Les propriétés CSS ont été mise entre parenthèse (pensez à faire une recherche sur le site mdn "nom de la propriété" + "mdn" pour comprendre comment elles fonctionnent)

  • bloc:1

    • Le texte doit être de couleur (propriété CSS "color") verte (green)
    • Le texte doit avoir une marge haute (propriété CSS "margin-top") de 2rem
    • Le texte doit avoir une couleur d'arrière-plan (propriété CSS "background-color") égale à "rgb(240, 255, 255)"

  • bloc:2

    • Tout élément ayant la classe "titre-principal" doit être de couleur (propriété CSS "color") violette (purple) et avoir une taille de texte (propriété CSS "font-size") de 1.35rem
    • Tout élément ayant la classe "description" doit avoir une marge interne gauche (propriété CSS "padding-left") de 1rem
    • Tout élément ayant la classe "nouvelle" doit avoir une marge externe basse (propriété CSS "margin-bottom") de 1.5rem
    • Tout élément ayant la classe "txt-gras" doit être gras (déclaration CSS "font-weight: bold")

Note : Les valeurs sont proposées à titre indicatif, vous pouvez bien évidemment mettre les valeurs qui vous plaisent. La documentation des propriétés CSS vous aidera à comprendre les valeurs attendues.

Je suis le texte exemple du TP

Bachelor universitaire de technologie en métiers du multimédia et de l'Internet

Cette formation orientée Web permet d’apprendre la programmation informatique, la conception de bases de données, le développement de contenus sur réseaux, la création numérique et sur mobiles (responsive web design). Elle implique une véritable compétence en gestion et encadrement de projet où la maîtrise de la communication, de l’audiovisuel et de l’infographie joue un rôle important.
