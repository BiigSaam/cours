Consignes de la pratique

  • Créer un fichier .css au même niveau que le fichier index.html appelé style.css
  • Lier le fichier .css à ce fichier index.html, grâce à la balise <link> (voir dans le <head> pour avoir un exemple)
  • Ecrire le contenu des attributs "style" dans le fichier style.css que vous venez de créer avec les sélecteurs de classe associés (vous devez impérativement créer des classes CSS si la balise n'en possède pas)
  • Supprimer ensuite les attributs "style" et leur valeurs
  • Ajouter une nouvelle balise, associez-lui une des classes que vous avez crée précédemment

L'attribut universel style contient des déclarations CSS afin de mettre en forme l'élément. Attention, il est recommandé de définir les règles de mise en forme dans un ou plusieurs fichiers séparés. Cet attribut, ainsi que l'élément <style> ont simplement pour but de permettre une mise en forme rapide, notamment pour tester.

Cette méthode n'est pas vraiment recommandée, ainsi qu'utiliser la balise <style>. On l'a vu en cours mais on arrêtera après.
