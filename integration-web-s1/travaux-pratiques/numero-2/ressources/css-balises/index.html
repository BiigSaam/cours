Pré-consignes de la pratique - facultatives (cliquer pour afficher / cacher)

  1. Créez un nouveau fichier html (avec la structure de base d'un site web. Rappel, vous avez le raccourci "html:5")
  2. Copiez la balise <main> (et son contenu de ce fichier) dans le fichier que vous venez de créer
  3. Sauvegardez-le et commencez votre TP dedans

Consignes de la pratique
  1. Ajouter l'attribut "style" à la balise <p> avec la valeur "color: red"
    • Sauvegardez le fichier (Ctrl/command (⌘) s) puis consultez votre page sur votre navigateur et pensez bien à actualiser votre page
  2. Ajouter sur la même balise, la déclaration "font-size: 2rem;"
    • Sauvegardez le fichier (Ctrl/command (⌘) s) puis consultez votre page sur votre navigateur et pensez bien à actualiser votre page
  3. Ajouter une balise <p> avec la même valeur pour l'attribut "style" que le balise <p> précédemment écrite
    • Sauvegardez le fichier (Ctrl/command (⌘) s) puis consultez votre page sur votre navigateur et pensez bien à actualiser votre page
