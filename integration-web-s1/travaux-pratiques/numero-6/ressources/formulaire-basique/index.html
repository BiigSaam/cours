Consignes de la pratique

(cliquer pour afficher)

Le but de cette pratique est de construire notre premier formulaire en y ajoutant au fur et à mesure des éléments de formulaire tels que la balise <input /> ou encore <select> Les formulaires sont des éléments clés du web 2.0 et 3.0, grâce à eux, et aux serveurs, l'utilisateur peut communiquer avec le site ou même d'autres internautes. De ce fait, il faut apporter beaucoup de soin à la construction des formulaires pour éviter de nuire à l'expérience de vos internautes. Sur un bon site, un internaute prend toujours du plaisir à y revenir.

  1. Partie 1

    • Ajouter une balise <form> avec la valeur "#" pour l'attribut "action"
    • Ajouter une balise <input> avec son label en prenant soin de rendre le tout accessible, et y entrer une valeur
    • Ajouter dans le même formulaire un autre champ input avec un type "intelligent" (exemple <input type="email">) et mettre une valeur incorrecte. Ne pas oublier le label associé avec l'attribut "for"
      • Voir documentation de la balise <label> (ouverture dans un nouvel onglet)
      • Voir documentation de la balise <input type="text"> (ouverture dans un nouvel onglet)
      • Voir documentation de la balise <form> (ouverture dans un nouvel onglet)

  2. Partie 2

    1. Rajoutez un élément de formulaire permettant sa soumission
    2. Mettez une valeur ("get" ou "post") pour la propriété "method"
    3. Soumettez le formulaire
      • Si target="GET" regardez l'url du navigateur, vous verrez les données dedans après soumission
      • Si target="POST" regardez l'onglet "console" de la console du navigateur ctrl + alt + i ou F12

  3. Partie 3

    1. Rajouter un textarea avec le label associé
    2. Rajouter des inputs type radio (min. 2)
    3. Rajouter des inputs type checkbox
    4. S'arranger pour que ça ressemble à quelque chose. Flexbox vous aidera.
      • Voir documentation des cases à cocher (ouverture dans un nouvel onglet)
      • Voir documentation des boutons radio (ouverture dans un nouvel onglet)
      • Voir documentation <textarea> (ouverture dans un nouvel onglet)

  4. Partie 4

    1. Rajouter une liste déroulante simple
    2. Rajouter une liste déroulante avec optgroup
      • Voir documentation de la balise <select> (ouverture dans un nouvel onglet)
      • Voir documentation de la balise <optgroup> (ouverture dans un nouvel onglet)
      • Accéder à un site pour pour générer une liste rapidement (ouverture dans un nouvel onglet)

  5. Partie 5

    1. Rajouter un pseudo-élément "::before" ou "::after" pour ajouter un "*" après tous les labels dont le champ est requis
      • Jouer sur la propriété "order" des enfants flexbox ou la valeur "row-reverse" de la propriété "flex-direction"
      • Jouer sur les combinateurs de voisin
      • Voir documentation de la propriété order (ouverture dans un nouvel onglet)
      • Voir documentation de la propriété "flex-direction" (ouverture dans un nouvel onglet)
      • Voir documentation du combinateur de voisin direct (ouverture dans un nouvel onglet)
