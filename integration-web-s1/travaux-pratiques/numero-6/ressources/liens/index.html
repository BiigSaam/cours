Consignes de la pratique

Partie 1 (cliquer pour afficher / cacher)
  1. Rajoutez une balise <a> permettant de faire une ancre au sein de cette page. N'oubliez pas l'attribut "href" dont la valeur sera précédée d'un croisillon (#)
    • Voir documentation de la balise <a> (ouverture dans un nouvel onglet)
    • Voir la documentation de la balise <a> d'un lien vers un élément de la même page (ouverture dans un nouvel onglet)
  2. Rajoutez une balise (de votre choix) et mettez-lui l'attribut id dont la valeur sera égale à la valeur de l'attribut "href" de la balise <a> que vous avez précédemment définie.
    Note : le défilement ne peut se voir que si votre page est assez haute. Rajoutez la classe "marge-haute" à votre balise cible (celle qu'on doit atteindre au clic) de votre ancre pour créer de la hauteur sur le site et donc voir le déplacement.
  3. Cliquez sur votre lien. Vous devriez atteindre la cible de votre ancre (la balise que vous venez de définir).

Partie 2 (cliquer pour afficher / cacher)
  1. Créez un nouveau fichier .html (avec le squelette d'une page) au même niveau que celui-ci
  2. Rajoutez un lien (balise <a>) dans ce fichier html
  3. Faites pointer (attribut href) ce lien vers le fichier que vous avez crée dans la première étape. La valeur de l'attribut "href" doit donc être égale au nom
  4. Cliquez sur votre lien. Vous devriez atteindre la page que vous venez de créer.

Partie 3 (cliquer pour afficher / cacher)
  1. Dans le fichier .html que vous avez crée dans la partie 2
    1. Rajoutez une image (relative ou absolue)
    2. Faites pointer cette image vers cette page html (index.html)
      Il faudra donc mettre votre balise <img> dans une balise <a>.
  2. Cliquez sur votre image. Vous devriez atteindre la page index.html.
