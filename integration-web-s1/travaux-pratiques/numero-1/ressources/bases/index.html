Consignes de la pratique

Partie 1 (cliquer pour afficher / cacher)

  1. Télécharger et installer le logiciel VS Code (nouvel onglet)
  2. Créez un fichier nommé "index.html" avec VS Code (File > New file)

  3. Ecrire "html:5" puis appuyez sur la touche "entrée"

    Vous devriez obtenir quelque chose comme ceci :

  4. Sauvegarder le fichier (Ctrl/command (⌘) s)
  5. Ouvrez le fichier dans votre navigateur préféré (pas internet explorer)
    • Glissez-déposez votre fichier index.html avec votre navigateur préféré (pas internet explorer)
    • Clic droit sur votre fichier index.html > ouvrir avec > votre navigateur préféré (pas internet explorer)
  6. Vous devriez obtenir une page blanche

Partie 2 (cliquer pour afficher / cacher)

  1. Rajouter une balise <p> avec le texte de votre choix au bon endroit dans le fichier index.html crée précédemment
  2. Sauvegarder le fichier (Ctrl/command (⌘) s) puis consultez votre page sur votre navigateur

Partie 3 (cliquer pour afficher / cacher)

  1. Rajouter plusieurs balises <p> avec le texte de votre choix à la suite de balise <p> écrite précédemment. (Raccourci shift alt / option ↑ ou ↓ pour dupliquer la ligne courante. Placer curseur textuel sur la bonne ligne.)
  2. Sauvegarder le fichier (Ctrl/command (⌘) s) puis consultez votre page sur votre navigateur

Partie 4 (cliquer pour afficher / cacher)

  1. Rajouter plusieurs balises consécutives <span></span> avec le texte de votre choix au bon endroit
  2. Sauvegarder le fichier (Ctrl/command (⌘) s) puis consultez votre page sur votre navigateur
