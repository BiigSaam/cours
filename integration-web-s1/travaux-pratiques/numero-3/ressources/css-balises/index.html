Pré-consignes de la pratique - facultatives (cliquer pour afficher / cacher)

  1. Créez un nouveau fichier html (avec la structure de base d'un site web. Rappel, vous avez le raccourci "html:5") dans le même dossier que celui-ci.
  2. Copiez les balises contenues entre les commentaires "<!-- Début: [...] -->" et "<!-- Fin: [...] -->"
    Il peut avoir plusieurs commentaires du même genre au sein du même fichier.
  3. Collez leur contenu dans votre même fichier aux mêmes endroits. Par exemple, si vous copiez du code dans le balise <head>, il faudra le coller également dans la balise <head> dans votre fichier.
  4. Sauvegardez-le et commencez votre TP dedans.

Consignes de la pratique
  1. Ajouter l'attribut "style" à la balise <p> avec la valeur "color: red"
    • Sauvegardez le fichier ( Ctrl/command (⌘) s ) puis consultez votre page sur votre navigateur et pensez bien à actualiser votre page
  2. Ajouter sur la même balise et dans la valeur de l'attribut "style", la déclaration "font-size: 2rem;"
    • Sauvegardez le fichier ( Ctrl/command (⌘) s ) puis consultez votre page sur votre navigateur et pensez bien à actualiser votre page
  3. Ajouter une balise <p> avec la même valeur pour l'attribut "style" que la balise <p> précédemment écrite
    • Sauvegardez le fichier ( Ctrl/command (⌘) s ) puis consultez votre page sur votre navigateur et pensez bien à actualiser votre page
