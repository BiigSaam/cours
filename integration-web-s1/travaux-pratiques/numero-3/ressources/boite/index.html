Consignes de la pratique

(cliquer pour afficher)

  1. Première partie :

    Dans le fichier css "style.css" rajouter le code suivant : "background-color: _couleur-que-vous-voulez_" dans la classe ".informations" déjà présente

    • documentation propriété background-color (ouverture dans un nouvel onglet)

  2. Seconde partie :

    Rajouter les propriétés "height" et "width" à la classe ".informations" avec une valeur

    • documentation propriété height (ouverture dans un nouvel onglet)
    • documentation propriété width (ouverture dans un nouvel onglet)

  3. Troisième partie :

    Rajouter la propriété "padding" avec une valeur numérique (suffixée de "px") supérieure à 60.

    • documentation propriété padding (ouverture dans un nouvel onglet)

  4. Quatrième partie :

    Annuler le padding-top et le padding-bottom de la balise <span> en mettant à zéro la valeur de ces propriétés et rajoutez les propriétés "padding-left" et "padding-right" avec les valeurs que vous souhaitez

    • Vous devez utiliser l'attribut "style" sur la balise <span>

  5. Cinquième partie :

    Ajouter la propriété "margin" à la classe "informations" avec la valeur que vous souhaitez (voir documentation)

    • documentation propriété margin (ouverture dans un nouvel onglet)

  6. Sixième partie :

    1. Rajouter une classe qui contient la propriété CSS "border" avec la valeur que vous souhaitez (voir documentation)
    2. Rajouter la classe fraîchement crée aux balises que vous souhaitez (celles dans la balise <main>)
      • documentation propriété border (ouverture dans un nouvel onglet)
