Consignes de la pratique

Partie 1 (cliquer pour afficher / cacher)

  1. Dans le fichier css "style.css", rajoutez la propriété CSS "background-color" avec une valeur appropriée pour faire un contraste dans la classe ".informations"

Partie 2 (cliquer pour afficher / cacher)

  1. Mettez les propriétés "height" et "width" à la classe ".informations" avec une valeur pour chaque propriété CSS

Partie 3 (cliquer pour afficher / cacher)

  1. Appliquez la propriété "padding" avec une valeur numérique (suffixée de "px") supérieure à 60
  2. Appliquez la propriété "padding" sur la balise <span>

Partie 4 (cliquer pour afficher / cacher)

  1. Ajoutez la propriété "margin" à la classe "informations" avec la valeur que vous souhaitez (voir documentation)

Partie 5 (cliquer pour afficher / cacher)

  1. Rajouter une classe qui contient la propriété CSS "border" avec la valeur que vous souhaitez (voir documentation)
  2. Rajouter la classe fraîchement crée aux balises que vous souhaitez (celles dans la balise <main>)
