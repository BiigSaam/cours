Consignes de la pratique

(cliquer pour afficher)

  1. Première partie :

    Dans le fichier css "style.css", rajoutez la propriété CSS "background-color" avec une valeur appropriée pour faire un contraste dans la classe ".informations"

  2. Seconde partie :

    Rajouter les propriétés "height" et "width" à la classe ".informations" avec une valeur

  3. Troisième partie :

    1. Rajouter la propriété "padding" avec une valeur numérique (suffixée de "px") supérieure à 60.
    2. Annuler le padding-top et le padding-bottom de la balise <span> en mettant à zéro la valeur de ces propriétés et rajoutez les propriétés "padding-left" et "padding-right" avec les valeurs que vous souhaitez.
      Note: Vous devez utiliser l'attribut "style" sur la balise <span>

  4. Quatrième partie :

    Ajouter la propriété "margin" à la classe "informations" avec la valeur que vous souhaitez (voir documentation)

  5. Cinquième partie :

    1. Rajouter une classe qui contient la propriété CSS "border" avec la valeur que vous souhaitez (voir documentation)
    2. Rajouter la classe fraîchement crée aux balises que vous souhaitez (celles dans la balise <main>)
