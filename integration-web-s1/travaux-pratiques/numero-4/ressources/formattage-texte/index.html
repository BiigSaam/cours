Consignes de la pratique (cliquer pour afficher / cacher)
  1. En utilisant une classe dédiée, changez l'alignement des paragraphes (différent pour chaque paragraphe)
    • Voir documentation propriété text-align (ouverture dans un nouvel onglet)
  2. Changez la "font-family" d'une partie du texte (utilisation d'une classe). Ne pas oublier les polices de secours.
    • Voir documentation propriété font-family (ouverture dans un nouvel onglet)
  3. Pour un fragment de texte : Souligner du texte et le rendre gras
    • Voir documentation propriété font-weight (ouverture dans un nouvel onglet)
    • Voir documentation propriété text-decoration (ouverture dans un nouvel onglet)
  4. Pour un fragment de texte : Augmentez la taille du texte et un autre fragment de texte : Diminuez la taille du texte
    • Voir documentation propriété font-size (ouverture dans un nouvel onglet)
  5. Pour un autre fragment de texte : Ajoutez une couleur d'arrière-plan
    • Voir documentation propriété background-color (ouverture dans un nouvel onglet)
