Pré-consignes de la pratique - facultatives (cliquer pour afficher / cacher)

  1. Créez un nouveau fichier html (avec la structure de base d'un site web. Rappel, vous avez le raccourci "html:5") dans le même dossier que celui-ci.
  2. Copiez les balises contenues entre les commentaires "<!-- Début: [...] -->" et "<!-- Fin: [...] -->"
    Il peut avoir plusieurs commentaires du même genre au sein du même fichier.
  3. Collez leur contenu dans votre fichier aux mêmes endroits. Par exemple, si vous copiez du code dans le balise <head>, il faudra le coller également dans la balise <head> dans votre fichier.
  4. Sauvegardez-le et commencez votre TP dedans.

Consignes de la pratique (cliquer pour afficher / cacher)
  • Pensez bien à utiliser une classe CSS dédiée pour chaque étape de la consigne
  1. En utilisant une classe CSS dédiée, changez l'alignement des paragraphes (différent pour chaque paragraphe)
    • Voir documentation propriété text-align (ouverture dans un nouvel onglet)
  2. Changez la "font-family" d'une partie du texte (utilisation d'une classe). Ne pas oublier les polices de secours.
    • Voir documentation propriété font-family (ouverture dans un nouvel onglet)
  3. Au sein d'un même paragraphe : Souligner et rendre gras une partie du texte
    • Voir documentation propriété font-weight (ouverture dans un nouvel onglet)
    • Voir documentation propriété text-decoration (ouverture dans un nouvel onglet)
  4. Au sein d'un même paragraphe : Augmentez la taille d'une partie du texte et pour une autre partie du texte : Diminuez la taille du texte
    • Voir documentation propriété font-size (ouverture dans un nouvel onglet)
  5. Pour une partie de texte : Ajoutez une couleur d'arrière-plan
    • Voir documentation propriété background-color (ouverture dans un nouvel onglet)
  6. Changer la sémantique (le balisage) de la partie "Listes" (celle sous la ligne) avec les bonnes balises
