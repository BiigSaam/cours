Pré-consignes de la pratique - facultatives (cliquer pour afficher / cacher)

  1. Créez un nouveau fichier html
  2. Sauvegardez le fichier (il faut impérativement qu'il ait l'extension ".html" autrement dit qu'il finisse pas ".html")
  3. Définissez la base d'une page html. Rappel : vous avez le raccourci "html:5"
  4. Copiez la balise <main> (et son contenu de la balise) dans le fichier html que vous venez de créer
  5. Copiez les balises <link> présentes dans le <head> dans le fichier html que vous venez de créer
  6. Sauvegardez-le et commencez votre TP dedans

Consignes de la pratique

Partie 1 (cliquer pour afficher / cacher)

  1. Dans le fichier CSS "style.css", rajoutez la propriété CSS "background-color" avec une valeur appropriée pour faire un contraste dans la classe ".informations". Aidez-vous du lien ci-dessous pour valider votre contraste.
    • Accéder à la documentation de la propriété "background-color" (ouverture dans un nouvel onglet)
    • Outil pour tester le constraste des couleurs (ouverture dans un nouvel onglet)

Partie 2 (cliquer pour afficher / cacher)

  1. Rajoutez les propriétés CSS "height" et "width" à la classe ".informations" avec une valeur pour chaque propriété CSS
    • Accéder à la documentation de la propriété "height" (ouverture dans un nouvel onglet)
    • Accéder à la documentation de la propriété "width" (ouverture dans un nouvel onglet)
  2. Qu'observez-vous ?

Partie 3 (cliquer pour afficher / cacher)

  1. Appliquez la propriété "padding" avec une valeur numérique et son unité sur la classe CSS "informations"
    • Accéder à la documentation de la propriété "padding" (ouverture dans un nouvel onglet)
  2. Qu'observez-vous ?

Partie 4 (cliquer pour afficher / cacher)

  1. Ajoutez la propriété "margin" à la classe CSS "informations" avec la valeur que vous souhaitez
    • Accéder à la documentation de la propriété "margin" (ouverture dans un nouvel onglet)
  2. Qu'observez-vous ?

Partie 5 (cliquer pour afficher / cacher)

  1. Rajoutez une bordure avec la propriété CSS "border" avec la valeur que vous souhaitez sur la classe CSS "informations" ou une classe CSS que vous venez d'ajouter
    • Accéder à la documentation de la propriété "border" (ouverture dans un nouvel onglet)

Partie 6 (cliquer pour afficher / cacher)

  1. Partant du principe que la classe "main-conteneur" a une largeur de 1 200px, arrangez-vous pour que chacune des balise <p> contenue occupe 50% du conteneur.
    • Note : il faudra utiliser la propriété "width".
      • Accéder à la documentation de la propriété "width" (ouverture dans un nouvel onglet)
    • Les deux éléments doivent avoir la même taille, utilisez la même classe CSS, c'est plus judicieux.
  2. Ajoutez un padding (ou une bordure assez large mini 10px) au moins latéral à la classe créée précédemment
  3. Qu'observez-vous ?

Partie 7 (cliquer pour afficher / cacher)

  1. Au début du fichier "style.css", ajoutez le sélecteur CSS suivant : 
    *, *::before, *::after {
    box-sizing: border-box;
}
  2. Et normalement tout devrait rentrer dans l'ordre. Jouez sur les valeurs des propriétés "padding" et "border" de la classe que vous venez d'ajouter, vos textes seront toujours côte à côte

Dans son cœur, HTML est un langage vraiment simple, composé d'éléments appliquables à des fragments de texte dans un document pour leur donner un sens différent (est-ce un paragraphe ? est-ce une liste à puces ? est-ce une partie de tableau ?), pour structurer un document en sections logiques (a‑t‑il un en-tête ? est-il sur trois colonnes ? a-t-il un menu de navigation ?) et pour intégrer du contenu comme images ou vidéos dans une page. Ce module est une introduction aux deux premiers concepts ; il présente les notions fondamentales et la syntaxe à connaître pour comprendre le HTML.
