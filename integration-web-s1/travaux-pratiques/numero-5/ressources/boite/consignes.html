Consignes de la pratique

Partie 1 (cliquer pour afficher / cacher)
  1. Ouvrez le fichier tp.html dans le navigateur. Qu'observez-vous ?

Partie 2 (cliquer pour afficher / cacher)
  1. Contrastez le texte du fichier tp.html grâce la propriété CSS "color" ou "background-color"
    • Accéder à la documentation de la propriété "background-color" (ouverture dans un nouvel onglet)
    • Accéder à la documentation de la propriété "color" (ouverture dans un nouvel onglet)

    N'oubliez pas de valider votre contraste avec la console du navigateur ou le site contrastchecker (ouverture dans un nouvel onglet)

Partie 3 (cliquer pour afficher / cacher)
  1. Créez un paragraphe (balise <p>) avec dedans une balise <span>
  2. Ajoutez sur ces deux balises la même classe CSS
  3. Créez un fichier CSS et ajoutez-y un sélecteur CSS correspondant à cette nouvelle classe avec dedans les propriétés CSS "height" et "width" avec une valeur pour chacune d'elle
    • Accéder à la documentation de la propriété "height" (ouverture dans un nouvel onglet)
    • Accéder à la documentation de la propriété "width" (ouverture dans un nouvel onglet)
  4. Qu'observez-vous ?

Partie 4 (cliquer pour afficher / cacher)
  1. Appliquez la propriété "padding" avec une valeur numérique et son unité sur la classe que vous avez crée précédemment
    • Accéder à la documentation de la propriété "padding" (ouverture dans un nouvel onglet)

Partie 5 (cliquer pour afficher / cacher)
  1. Créez une nouvelle balise de type bloc (<div>, <p>, <section>...) avec une nouvelle classe qui possède la propriété CSS margin. Les marges doivent être différentes pour chaque côté
    • Accéder à la documentation de la propriété "margin" (ouverture dans un nouvel onglet)
  2. Qu'observez-vous ?

Partie 6 (cliquer pour afficher / cacher)
  1. Rajouter une bordure sur une des classes que vous avez crée précédemment
    • Accéder à la documentation de la propriété "border" (ouverture dans un nouvel onglet)
