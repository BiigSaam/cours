Consignes de la pratique

Partie 1 (cliquer pour afficher / cacher)

  1. Ajoutez le code nécessaire pour réaliser un lien interne.
    Note : le défilement ne peut se voir que si votre page est assez haute. N'hésitez pas à mettre une marge ou plein de contenus
    • Voir documentation de la balise <a> (ouverture dans un nouvel onglet)

Partie 2 (cliquer pour afficher / cacher)

  1. Créez un nouveau fichier .html
  2. Rajoutez un lien dans le premier fichier html
  3. Faites pointer ce lien vers le fichier que vous avez crée dans la première étape
  4. Dans le fichier html que vous avez crée précédemment, faites pointer une image vers le premier fichier html
