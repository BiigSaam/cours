  1. Partie 1

    • Éditer la règle @media de façon à ce que le titre "Ma galerie d'images" plus gros sur un appareil dont la largeur d'écran est inférieure ou égale à 320px
    • Ajouter une règle @media pour gérer un écran dont la taille est supérieure ou égale à 400px et appliquer des déclarations CSS sur n'importe quel sélecteur CSS
    • Ajouter une règle @media pour gérer un écran ayant l'orientation paysage (landscape en anglais) et appliquer des déclarations CSS sur n'importe quel sélecteur CSS

  2. Partie 2

    1. Lister le nombre de règles @media contenues dans le fichier style.css
    2. Créer autant de fichiers .css qu'il y a de règles @media qui ont été écrites dans le fichier style.css dans la partie 1
    3. Lier le ficher "index.html" aux fichiers css (ne pas oublier les attributs media)
    4. Placer le contenu des règles @media dans les bons fichiers

Ma galerie d'images

Tous les jours, une demi-heure de reportage sans commentaire. Inspirés par la célébrissime émission de radio américaine This American Life, "Les Pieds sur Terre" s’organisent désormais autour de récits, d’histoires vraies, une, deux ou trois par émission, qui tournent autour d’un même thème. Ces histoires sont racontées à la première personne et nourries d’éléments de reportage.

Si deux valeurs grid-line sont utilisées pour la propriété, la valeur de grid-column-start sera définie avec la première valeur avant la barre oblique et la valeur de grid-column-end sera définie avec celle qui est située après la barre oblique.
Un mot-clé qui indique que la propriété ne contribue pas au placement de l'élément sur la grille. Cela indique un placement automatique, une taille de fragment (span) automatique ou une taille par défaut de 1.
