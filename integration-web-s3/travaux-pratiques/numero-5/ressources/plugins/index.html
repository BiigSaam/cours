Partie 1 (cliquer pour afficher / cacher)

  1. Chargez la bibliothèque lodash. Utilisez une de ces fonctions (voir documentation pour voir des exemples à copier dans votre code) et affichez le résultat dans la console du navigateur.
    • Accéder au site de lodash (ouverture dans un nouvel onglet)
    • N'hésitez pas à tester d'autres fonctions. Lodash est une bibliothèque javascript très puissante et utile pour manipuler les données (tableaux et objets).

Vous pouvez télécharger le plugin ou utiliser le lien CDN pour charger le script à distance. Dans tous les cas, il faudra utiliser la balise <script> pour charger le script, et n'oubliez pas l'attribut "defer" pour charger le script après le chargement des balises.

Partie 2 (cliquer pour afficher / cacher)

  1. Suivez le didacticiel de chartjs (ouveture dans un nouvel onglet) et essayez d'afficher quelques graphiques. Une fois suffisamment à l'aise avec la bibliothèque passez à la partie suivante.

Partie 3 (cliquer pour afficher / cacher)

  1. Utilisez les données du fichier "donnees-courbe.js" et affichez une courbe. Le fichier contient l'évolution des naissances du prénom Pauline de 1900 à nos jours
    • Voir documentation du graphique en bâtons de chartjs (ouverture dans un nouvel onglet)
    • Si vous voulez conserver
    • Astuce : pour charger les données, vous n'avez qu'à charger le fichier "donnees-courbe.js" avant votre script, ainsi vous pourrez accéder à la variable "prenomPaulineEvolution" qui contient les données.
    • Astuce 2 : pour utiliser ces données, il faudra les formater, voici le code qu'il faudra utiliser : 
                      const data = prenomPauline.map((item) => item.nombre)
        const listeAnnees = prenomPaulineEvolution.map((item) => item.annais)
      Ensuite, il faudra les utiliser pour les bons paramètres pour afficher le graphique.
  2. Utilisez les données du fichier "donnees-batons.js" et affichez un graphique en bâtons
    • Voir documentation du graphique en bâtons de chartjs (ouverture dans un nouvel onglet)
    • Le fichier contient toutes les naissances du prénom "Jérôme" en Ile-de-France durant l'année 2005
    • Astuce : pour charger les données, vous n'avez qu'à charger le fichier "donnees-batons.js" avant votre script, ainsi vous pourrez accéder à la variable "prenomJerome2005" qui contient les données.
    • Astuce 2 : pour utiliser ces données, il faudra les formater, voici le code qu'il faudra utiliser : 
                      const data = prenomJerome2005.map((item) => item.nombre)
        const labels = prenomJerome2005.map((item) => item.dpt)
      Ensuite, il faudra les utiliser pour les bons paramètres pour afficher le graphique.

Pour aller plus loin (cliquer pour afficher / cacher)

  1. Réutilisez le code utilisé dans la partie 2 pour afficher les mêmes graphiques que précédemment, à la différence près que les données ne doivent pas provenir des fichiers "donnees-batons.js" et "donnees-courbe.js" mais des fichiers "jerome.json" et "pauline.json" (présents dans la ressource)
