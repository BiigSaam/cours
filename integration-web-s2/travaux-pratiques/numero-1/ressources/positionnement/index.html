Pré-consignes de la pratique - facultatives (cliquer pour afficher / cacher)

  1. Créez un nouveau fichier html (avec la structure de base d'un site web. Rappel, vous avez le raccourci "html:5")
  2. Copiez la balise <main> (et son contenu) dans le fichier HTML que vous venez de créer (si elle existe)
  3. Copiez les balises <link> présentes dans le <head> dans le fichier html que vous venez de créer
  4. Sauvegardez-le et commencez votre TP dedans

Positionnement

Position relative : L'élément est positionné dans le flux normal du document puis décalé, par rapport à lui-même, selon les valeurs fournies par top, right, bottom et left. Le décalage n'a pas d'impact sur la position des éléments. Aussi, l'espace fourni à l'élément sur la page est le même que celui fourni avec static.

Position absolute : Un élément positionné de façon absolue est un élément dont la propriété de position calculée est absolute ou fixed. Dans ce cas, les propriétés top, bottom, right et left indiquent les distances entre les bords de l'élément et les bords du bloc englobant (c'est-à-dire l'ancêtre par rapport auquel l'élément est positionné). Si l'élément possède des marges, elles sont ajoutées aux décalages.

Position fixed : La plupart du temps, les éléments positionnés de façon absolue ont leurs propriétés height et width qui valent auto afin que le contenu ait suffisamment d'espace. Toutefois, les éléments positionnés de façon absolue et qui ne sont pas des éléments remplacés peuvent remplir l'espace vertical en utilisant top et bottom tout en laissant height indéfini (c'est-à-dire avec la valeur auto). On pourra faire de même pour l'espace horizontal avec les propriétés left et right.
