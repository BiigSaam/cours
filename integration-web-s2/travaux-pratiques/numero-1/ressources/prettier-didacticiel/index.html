Tutoriel activation prettier

    1. Cliquer sur l'écrou en bas à gauche de votre fenêtre VS Code
    1. Ecrire "defaultFormatter" dans la zone de texte
    2. Sélectionner "esbenp.prettier-vscode" dans la liste déroulante
    3. Ça y est, Prettier est maintenant actif sur tous les fichiers eligibles au formattage

Finissez le TP

    1. Retourner dans le fichier index.html de la ressource "mal-indente"
    2. Appuyer sur Alt + Maj + F
    3. Votre code devrait être bien indenté maintenant
    4. Faites la même chose avec votre code CSS
    1. Formatter le fichier "mal-indente/erreur.css"
    2. Et il ne se passe rien !
