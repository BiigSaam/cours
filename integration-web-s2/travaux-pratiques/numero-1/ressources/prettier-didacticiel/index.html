TP activation extension Prettier pour VSCode

  1. Cliquer sur l'écrou en bas à gauche de votre fenêtre VS Code puis l'option "Settings" ("Préférences" en français)
  2. Ecrire "defaultFormatter" dans la zone de texte
  3. Sélectionner "esbenp.prettier-vscode" dans la liste déroulante
  4. Ça y est, Prettier est maintenant actif sur tous les fichiers eligibles au formattage

Finissez le TP

    1. Retourner dans le fichier index.html de la ressource "mal-indente"
    2. Appuyer sur Alt + Maj + F (Windows/MacOS/Linux)
    3. Votre code devrait être bien indenté maintenant
    4. Faites la même chose avec le code CSS du fichier "style.css" de la même ressource
