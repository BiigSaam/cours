Pré-consignes de la pratique - facultatives (cliquer pour afficher / cacher)

  1. Créez un nouveau fichier html (avec la structure de base d'un site web. Rappel, vous avez le raccourci "html:5")
  2. Copiez la balise <main> (et son contenu) dans le fichier HTML que vous venez de créer (si elle existe)
  3. Copiez les balises <link> présentes dans le <head> dans le fichier html que vous venez de créer
  4. Sauvegardez-le et commencez votre TP dedans

Consignes de la pratique

Partie 1 (cliquer pour afficher / cacher)
  1. Survolez les éléments présents après les consignes
    • Note : si les couleurs du lien ne vous plaisent pas n'hésitez pas à les changer dans le fichier css de la ressource

Partie 2 (cliquer pour afficher / cacher)
  1. Appliquez une transition sur la propriété "color" sur la classe ".lien-moodle"
    • Voir documentation de la propriété transition (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété color (ouverture dans un nouvel onglet)
  2. Survolez le lien, que constatez-vous ?
    • Une fois cette consigne effectuée, essayez de placer la propriété CSS "transition" (et sa valeur) dans la pseudo-classe ":hover" de la classe "lien-moodle"

Partie 3 (cliquer pour afficher / cacher)
  1. Effectuez la même chose que l'étape précédent mais sur la propriété "margin-top", "margin-bottom", "margin-left" ou "margin-right"
    • Voir documentation de la propriété margin-right (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété margin-left (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété margin-top (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété margin-bottom (ouverture dans un nouvel onglet)

Partie 4 (cliquer pour afficher / cacher)
  1. Ajoutez un nouveau sélecteur CSS permettant d'effectuer une transition sur le lien (celui utilisé précédemment) et le paragraphe situé sous ce lien
    • Ce nouveau sélecteur CSS doit impérativement impliquer le sélecteur CSS "conteneur", il vous faudra donc jouer sur l'imbrication de sélecteurs CSS (voir exemple dans la présentation du cours associé)
  2. Survolez la balise .conteneur pour valider votre code

Partie 5 (cliquer pour afficher / cacher)
  1. Ajoutez une nouvelle balise (vous pouvez copier la balise section ayant la classe "positionnement") en position absolue
  2. Retirez toute classe différente de "positionnement" (si vous avez copié la balise section du début)
  3. Arrangez-vous pour que la balise chevauche de l'autre balise en position absolue créée précédemment
    • Il faut utiliser les propriétés "top", "bottom", "left" ou "right" pour déplacer la balise pour rappel
  4. Utilisez la propriété "z-index" pour changer le chevauchement des balises
    • Voir documentation de la propriété z-index (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété position (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété top (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété bottom (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété left (ouverture dans un nouvel onglet)
    • Voir documentation de la propriété right (ouverture dans un nouvel onglet)
