Explications

Le but de ce mini-projet est de développer dans une des fonctionnalités javascript attendue pour la SAE 203 : Cacher la bannière affichée après soumission du formulaire au clic sur un bouton. Pour ce faire, nous aurons donc besoin de deux éléments : une balise <button type "button"> et une balise contenant notre bannière. Une fois ces deux éléments ajoutés, nous allons rajouter des data-attributes à chacun de ses éléments pour pouvoir y accéder dans le javascript.

Un bouton ne doit jamais être remplacé par la balise <a> pour appeler une fonction au clic. Si vous devez appeler une fonction au clic sur un élément, il faut utiliser la balise <button type="button">.

Rappel - data-attributes : Ces sont des attributs personnalisés que vous pouvez mettre sur n'importe quelle balise HTML, le nombre de data-attributes sur une balise est infini. Vous pouvez les nommer comme vous le souhaitez. Un data-attribut doit impérativement commencer par "data-" et ne pas contenir d'espaces. Un data-attribute peut également contenir une valeur qui ne peut être qu'une chaîne de caractères ou un nombre.

  • Voir documentation des data-attributes

Maintenant que nous vous avons mis en place notre structure HTML, nous pouvons passer au javascript. Nous avons donc crée un fichier avec une extension ".js" et chargé dans cette page HTML grâce à la balise <script>.

L'attribut "defer" dans la balise <script> permet de dire au navigateur de charger le script une fois que toutes les balises HTML sont chargées. En utilisant cet attribut, vous pouvez mettre vos balises <script> avant la fermeture de la balise <body> ou dans la balise <head>.

Vous trouverez des explications sur le code javascript dans le fichier "script.js".
