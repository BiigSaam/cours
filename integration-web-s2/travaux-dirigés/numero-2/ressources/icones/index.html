Consignes de la pratique - Partie 1

  • Aller sur le site de génération de favicon (nouvel onglet)
  • Sélectionner une image (format carré et taille supérieure à 250px de largeur)
  • Générer le favicon (Cliquer sur "Generate favicon")
  • Copier tout le code généré (il commence par "<link>") (oui, il y a beaucoup de lignes)
  • Télécharger l'archive générée par le site (Cliquer sur "Download the generated favicon")
  • Déarchiver l'archive
  • Coller tout le contenu du dossier à la racine de la ressource

Utilisation de icomoon

Afficher / cacher étapes pour gérer les icônes
  • Aller sur le site icomoon.io (nouvel onglet)
  • A : Permet d'importer des icones personnalisées. Format .svg ou .json
  • B : Permet de sélectionner des des icônes proposées par icomoon
  • C : Boite à outils
  • A : Affiche les préférences de notre iconfont
  • B : Affiche le contenu de notre iconfont
    • B1 : icône
    • B2 : nom de l'icône (modifiable)
    • B3 : valeur du caractère
  • C : Permet de télécharger notre iconfont (et tout ce qui va avec)
  • D : Permet de retourner à l'étape précédente

Consignes de la pratique - Partie 2

  • Aller sur le site icomoon.io (nouvel onglet)
  • Sélectionner des icônes parmi celles proposées par icomoon (8-9)
  • Télécharger l'iconfont (voir explications dans "Afficher / cacher étapes..."
  • (Désarchiver et) Déplacer le contenu de l'archive dans la ressource
  • Lier le fichier .css à la ressource
  • Afficher une icône à côté de chaque champ de formulaire
  • Afficher une icône dans le texte et bien l'aligner
  • Ajouter des transitions au hover (couleur, rotation, échelle...) sur les icônes
    • Voir la documentation de la pseudo-classe ":hover" (nouvel onglet)
    • Voir la documentation des transitions css (nouvel onglet)

Consignes de la pratique - Partie 3

  • Télécharger une icône sur le site iconmonstr (nouvel onglet) (désactivez votre bloqueur de pub)
  • Télécharger, au format svg, une icône de flèche vers le bas (arrow en anglais)
  • Télécharger, au format svg, une icône lié à l'envoi (avion en papier par exemple / paper plane en anglais)
  • Retourner sur le site icomoon.io
  • Importer vos fichiers svg et le fichier selection.json
  • Sélectionner les icônes dont vous avez besoin (garder celles que vous avez déjà)
  • Télécharger la nouvelle version de votre iconfont
  • Mettre en place l'icône de flèche sur le select (position:absolute) et l'icône d'envoi dans le bouton "Envoyer"
