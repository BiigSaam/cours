Le web scraping (parfois appelé harvesting) est une technique d'extraction du contenu de sites Web, via un script ou un programme, dans le but de le transformer pour permettre son utilisation dans un autre contexte, cette technique peut être très utile pour les data-journalistes.

Dans le cadre de ce cours, nous utilisons le langage de programmation Python, un langage très puissant mais très simple à prendre en main pour les non-initiés à la programmation, ceci a ses avantages et ses inconvénients.

Le Python est notamment utilisé en data-science ou encore dans l'UI du système d'exploitation Ubuntu.
