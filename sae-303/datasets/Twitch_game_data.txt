- Rank : classement
- Game : Nom du jeu
- Month : Mois (1 - 12)
- Year : Année
- Hours_watched : Nombre d'heures visionnées
- Hours_Streamed : Nombre d'heures streamées
- Peak_viewers : Nombre max de vues
- Peak_channels : Nombre max de chaînes jouant au jeu
- Streamers : Nombre moyen de streamers sur le jeu
- Avg_views : nombre de vues moyennes
- Avg_channels : Nombre moyens d'abonnés présents
- Avg_viewer_ratio : Nombre de vues moyen
