Consignes de la pratique - TP3 - Balise <template>

Partie 1 (cliquer pour afficher / cacher)
  1. En vous inspirant de l'exemple dans les slides du cours, créez et affichez votre propre template à un fichier HTML que vous aurez créer.

N'oubliez pas de créer un fichier javascript pour charger puis afficher votre template. Et bien évidemment de cloner votre template avant de l'utiliser.

Partie 2 (cliquer pour afficher / cacher)
  1. Arrangez-vous pour modifier le contenu d'une ou plusieurs balises de votre template depuis le javascript.
    • Cette modification peut être de différent ordre : texte, lien d'image...

Si il est préférable d'utiliser un data-attribute pour cibler un élément. Si votre template est relativement simple (des balises uniques), vous pouvez cibler directement la balise, mais attention si choix de balise n'est pas sûr.

Partie 3 (cliquer pour afficher / cacher)
  1. Utilisez la fonction "getListNationalHolidays" contenue dans le fichier index.js
  2. Affichez son contenu grâce à un template. Ceci peut être une liste ou un tableau

L'API retourne une liste.
